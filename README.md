# mavericks-typora

A dark color theme for [Typora](https://typora.io/) inspired by the ocean and big waves.

Part of the [Mavericks theme](https://github.com/nickrotondo/mavericks-theme) collection.

## Install

1. Open Typora and go to **Preferences** > **Appearance** > **Open Theme Folder**
2. Copy `mavericks.css` and the `mavericks/` folder into the themes directory
3. Restart Typora
4. Select **Mavericks** from the **Themes** menu
